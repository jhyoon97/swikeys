# 스위치 리서치 Skill

사용자가 특정 제조사의 스위치를 리서치해달라고 요청하면 이 절차를 따릅니다.

## 절차

1. **웹 리서치 (다단계 수집)**: 아래 소스들을 **모두** 체크하여 최대한 빠짐없이 수집합니다. 한 소스에서만 검색하면 누락이 발생하므로, 반드시 여러 소스를 교차 확인해야 합니다.

   **1단계: 스위치 데이터베이스/카탈로그 사이트** (가장 포괄적인 목록 확보)
   - [kbd.news](https://kbd.news) — 제조사별 스위치 목록 검색
   - [keeb-finder.com](https://keeb-finder.com) — 스위치 카탈로그 검색
   - [theremingoat.com](https://theremingoat.com) — 스위치 리뷰 아카이브
   - [switches.mx](https://switches.mx) — 스위치 스펙 DB (있을 경우)

   **2단계: 주요 해외 리테일러** (판매 목록에서 누락분 보충)
   - [divinikey.com](https://divinikey.com) — `{제조사}` 검색
   - [kineticlabs.com](https://kineticlabs.com) — `{제조사}` 검색
   - [milktooth.com](https://milktooth.com) — `{제조사}` 검색
   - [unikeyboards.com](https://unikeyboards.com) — `{제조사}` 컬렉션 페이지
   - [mechanicalkeyboards.com](https://mechanicalkeyboards.com) — `{제조사}` 검색
   - [cannonkeys.com](https://cannonkeys.com) — `{제조사}` 검색
   - [zkeebs.com](https://zkeebs.com) — `{제조사}` 검색

   **3단계: 한국 리테일러** (국내 유통 제품 확인)
   - 스웨그키, 펀키스, 폼포키, 지온웍스 등 국내 판매처 검색

   **4단계: 교차검증 및 누락 확인**
   - 1~3단계에서 수집한 전체 스위치 이름 목록을 정리
   - `"{제조사} switches"`로 추가 웹 검색하여 누락 여부 최종 확인
   - 콜라보 제품은 콜라보 파트너명으로도 검색 (예: `"Sillyworks x HMX"`, `"XL Lab HMX"`)
   - 수집된 스위치 수가 해당 제조사 규모 대비 적다고 판단되면 추가 검색 수행

   **5단계: 한글 조사**
   - 영문 조사와 별개로, 한글로도 동일한 리서치를 수행
   - `"{제조사 한글명} 스위치"`로 검색하여 영문 조사에서 누락된 스위치가 있는지 확인
   - 한국 판매처/커뮤니티에서 공식 한글 제품명이 확인되면 기재, 없으면 `-`로 표기

2. **결과 정리**: `docs/{제조사명}-switches-research.md` 파일에 아래 포맷으로 정리합니다.

3. **표 컬럼**: `src/types/switch.ts`의 `KeyboardSwitch` 인터페이스를 기준으로 하되, 아래 컬럼을 사용합니다:

| 컬럼명 | 설명 | 비고 |
|--------|------|------|
| 이름 | 스위치 전체 이름 (영어) | |
| 한글이름 | 스위치 한글 이름 (제조사 포함) | 공식 한글명이 있는 경우만 기재, 없으면 `-` |
| 제조사 | 스위치를 **생산하는 공장/회사** | 콜라보 파트너 제외 |
| 콜라보업체 | 디자인/기획 파트너 | 없으면 `-` |
| 스위치타입 | `리니어`, `택타일`, `hall effect` 중 택1 | |
| 저소음 | `Yes` / `No` | 사일런트 스위치 여부 |
| 로우프로파일 | `Yes` / `No` | 로우프로파일 스위치 여부 |
| 상부하우징재질 | | |
| 하부하우징재질 | | |
| 스템재질 | | |
| 공장윤활 | `Yes` / `No` / `Partial` | |
| 스프링길이(mm) | | 미확인 시 `-` |
| 마운트핀 | `5` / `3` | |
| 트래블(mm) | 총 트래블 거리 | |
| 입력지점(mm) | 액추에이션 포인트 | |
| 입력압(g) | 액추에이션 포스 | |
| 초기압(g) | 이니셜 포스 | |
| 바닥압(g) | 바텀아웃 포스 | |
| 걸림압(g) | 택타일/클릭키 전용 — 택타일 범프 피크 포스 | 택타일·클릭키 섹션 테이블에만 포함, 리니어 섹션에는 해당 컬럼 없음 |
| 출처 | 정보 출처 URL (plain URL) | 마크다운 링크 아님 |

4. **표 분류**: 스위치 타입별로 섹션을 나누어 정리합니다:
   - 리니어 스위치: 19컬럼 (걸림압 컬럼 없음)
   - 택타일 스위치: 20컬럼 (`걸림압(g)` 컬럼이 `바닥압(g)` 뒤, `출처` 앞에 위치)
   - 클릭키 스위치: 20컬럼 (택타일과 동일하게 `걸림압(g)` 포함)
   - 사일런트 스위치는 별도 섹션 없이 `저소음` 컬럼에서 `Yes`로 구분

5. **제조사 vs 콜라보업체 구분 규칙**:
   - **제조사**: 스위치를 물리적으로 생산하는 회사 (예: HMX, Gateron, Cherry 등)
   - **콜라보업체**: 디자인, 기획, 커스텀 사양을 담당한 파트너 (예: Sillyworks, Kinetic Labs 등)
   - 이름에 `HMX x Partner` 형태로 표기된 경우 `Partner`가 콜라보업체
   - 콜라보업체가 여러 곳이면 쉼표로 구분 (예: `SPM, Soulcat`)

6. **데이터 규칙**:
   - 스위치 스펙을 100% 완벽하게 채울 필요는 없습니다. 일부 스펙이 비어있더라도 스위치 이름과 기본 정보(제조사, 타입 등)가 확인되면 포함합니다. 스펙이 불완전하다는 이유로 스위치를 누락시키지 마세요.
   - 확인할 수 없는 값은 `-`로 표기
   - **Hall Effect 스위치는 `입력압`과 `입력지점`을 항상 `-`로 비워둡니다.** Hall Effect 스위치는 소프트웨어로 입력지점을 자유롭게 설정할 수 있어 고정된 입력압/입력지점 값이 의미가 없습니다.
   - 무게 변형(Light/Heavy 등)은 별도 행으로 분리
   - 출처는 plain URL로 기재 (마크다운 링크 문법 사용하지 않음)
   - **한글이름**: 한국 판매처(스웨그키, 지온웍스, 펀키스, 폼포키 등)에서 공식 한글 제품명이 확인되는 경우에만 기재. 억지로 영어를 음역하지 않으며, 한글명이 없으면 `-`로 표기
